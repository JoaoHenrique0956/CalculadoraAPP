# 📱 Calculadora Mobile (Kotlin)

Este projeto consiste no desenvolvimento de uma **calculadora mobile** utilizando **Kotlin** no **Android Studio**.

## ✨ Funcionalidades

A aplicação realiza operações matemáticas básicas com dois ou mais números:

* ➕ **Soma**
* ➖ **Subtração**
* ✖️ **Multiplicação**
* ➗ **Divisão**

## 🚀 Tecnologias Utilizadas

* Kotlin
* Android Studio
* XML (Layout)
